# mymap
OSM-based map painter and visits keeper

== To start ==

* cd app/
* npm install express
* npm install body-parser
* /* prepare google-config.txt */
* run.sh app.js

* cd api/
* go get github.com/gorilla/mux
* go get github.com/gorilla/handlers
* go build
* api/api

* $browser http://localhost:8081
